ORI(1)                        User Commands                        ORI(1)

NAME
       ori — 프론트엔드 개발자 (duck edition)

SYNOPSIS
       ori [--build <재미있는것>] [--coffee[=refill]] [--until=새벽]

DESCRIPTION
       "돌아간다"와 "잘 만들었다"의 간격을 좁히는 일을 합니다.
       입력은 커피, 출력은 인터페이스입니다.

OPTIONS
       --pixel-perfect
              기본값입니다. 끄는 방법은 아직 발견되지 않았습니다.

       --coffee[=refill]
              연료를 주입합니다. 중첩 호출이 허용됩니다.

       --until=새벽
              집중 모드. 해가 뜨면 자동으로 SIGSLEEP을 수신합니다.

EXIT STATUS
       0      오늘도 무사히 커밋했습니다.

BUGS
       새벽에 작성한 코드가 아침에 다르게 읽히는 현상이 보고되어
       있습니다. 재현 조건이 명확하여 wontfix 처리되었습니다.

SEE ALSO
       portfolio(1), blog(7), mail(1) — 상단 버튼 참조

2026-07                       ori terminal                         ORI(1)
